## 功能特性

### 图表类型
- **折线图**：展示趋势数据
- **柱状图**：比较分类数据
- **饼图**：显示比例关系
- **面积图**：展示累计数据
- **散点图**：分析相关性

### 交互功能
- **一键切换**：支持图表类型快速切换
- **缩放平移**：支持图表区域缩放和平移操作
- **hover详情**：鼠标悬停显示详细数据信息
- **图例显隐**：可控制图例显示和隐藏
- **区域选择**：支持数据区域高亮选择

### 数据管理
- **实时编辑**：可直接在图表旁边编辑数据
- **多维度筛选**：支持时间、类别、条件筛选
- **数据导入导出**：支持Excel和PNG格式导出
- **基准线对比**：添加基准线进行数据对比

### 视觉效果
- **动画效果**：图表切换和数据更新带有平滑动画
- **多图联动**：多个图表之间数据联动
- **样式自定义**：支持颜色、主题自定义
- **数据钻取**：支持层级数据钻取分析

### 高级功能
- **自动刷新**：支持定时数据自动刷新
- **异常提醒**：数据异常自动提醒
- **全屏展示**：支持全屏模式查看
- **响应式适配**：适配不同设备屏幕
- **操作撤销/重做**：支持操作历史管理

## 技术栈

- **前端框架**：纯HTML/CSS/JavaScript
- **图表库**：Chart.js
- **响应式设计**：CSS Grid + Flexbox
- **动画效果**：CSS3动画 + JavaScript动画

## 使用说明

### 基本操作
1. **选择图表类型**：在左侧工具栏选择需要的图表类型
2. **编辑数据**：点击图表旁边的数据表格进行编辑
3. **调整样式**：使用右侧样式面板调整颜色和样式
4. **导出数据**：点击导出按钮保存数据或图片

### 高级功能
- **数据联动**：多个图表选择相同数据集实现联动
- **筛选功能**：使用筛选器对数据进行多维度筛选
- **动画设置**：在设置中调整动画效果和速度
