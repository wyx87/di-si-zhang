# 数据可视化交互平台

一个基于HTML5、CSS3和JavaScript的交互式数据可视化平台，包含6个不同类型的图表模块和主题切换功能。

## 🌟 项目特色

### 📊 数据可视化模块
1. **图书采购情况柱状图** - 两个地区图书采购数据对比，支持实时数据编辑
2. **汇率走势折线图** - 2017年和2019年美元/人民币汇率走势对比
3. **三角函数图像** - 可调节振幅和函数类型的三角函数可视化
4. **基础柱状图实例** - 简单的类别数据展示和编辑
5. **数据分布饼图** - 占比分布展示
6. **散点分布图** - 坐标点分布关系展示

### 🎨 主题切换功能
- 6种精美主题：淡色、蓝色、绿色、紫色、橙色、深色
- 实时颜色切换，所有图表颜色同步更新
- 本地存储记忆用户选择的主题

### 💡 交互功能
- 实时数据输入编辑
- 参数调节滑块
- 图表类型切换
- 响应式设计，适配各种设备
- 平滑的动画效果

## 🛠️ 技术栈

- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **图表库**：Chart.js
- **主题系统**：CSS变量和类切换
- **数据存储**：localStorage

## 📋 功能详细说明

### 1. 图书采购情况柱状图
- 实时编辑两个地区的图书采购数据
- 支持家庭类、小说类、心理类、科技类、儿童类数据
- 地区间数据对比可视化

### 2. 汇率走势折线图
- 2017年和2019年7月汇率数据对比
- 可编辑每日汇率数据
- 折线图展示趋势变化

### 3. 三角函数图像
- 选择正弦、余弦、正切函数
- 滑块调节振幅参数
- 实时更新函数图像

### 4. 基础柱状图
- 简单类别数据展示
- 实时数据编辑功能
- 基础柱状图模板

### 5. 数据分布饼图
- 占比分布可视化
- 支持多项目数据编辑
- 饼图展示各部分比例

### 6. 散点分布图
- 坐标点分布关系
- 可编辑X、Y坐标数据
- 散点图展示数据分布

## 🎯 使用指南

### 基本操作
1. 点击图表下方的数据输入框编辑数据
2. 点击"更新图表"按钮应用修改
3. 使用右上角主题选择器切换颜色主题
4. 三角函数使用下拉菜单选择函数类型和滑块调节振幅

### 主题切换
- 点击右上角主题按钮
- 选择6种预设主题之一
- 所有界面元素会自动适配新主题
- 选择的主题会自动保存

## 🔧 开发说明

### 添加新主题
在CSS中修改主题变量可以添加新主题：

```css
.new-theme {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --background-color: #your-color;
    --card-bg: #your-color;
    --text-color: #your-color;
    --border-color: #your-color;
}
```

### 添加新图表模块
1. 在HTML中添加新的图表容器
2. 在JavaScript中添加对应的图表初始化函数
3. 更新数据更新逻辑
